Unified Academic Hub is a client-side academic portal designed for IIIT Surat-style classroom 
workflows. The application provides role-based authentication, subject enrollment, course 
material sharing, assignment creation and submission, announcement posting, attendance 
tracking, marks management, and direct messaging in a single responsive dashboard. Browser 
localStorage is used to persist users, subjects, enrollments, resources, submissions, messages, 
attendance, and marks without requiring a backend server in the current prototype. 
The project demonstrates practical knowledge of semantic HTML structure, modular CSS 
design, JavaScript event handling, DOM manipulation, FileReader-based uploads, JSON-based 
local data storage, responsive layouts, dark/light theme handling, and CRUD operations. It is 
designed as a functional proof of concept that can later be extended with a secure backend, 
database, cloud synchronization, analytics, and native mobile application support.
